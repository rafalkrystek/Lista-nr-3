# Prosta Aplikacja i Testy Jednostkowe

## Opis

Ten projekt zawiera prostą aplikację napisaną w Pythonie oraz testy jednostkowe dla tej aplikacji. Dodatkowo projekt zawiera plik Makefile, który umożliwia instalację zależności, uruchamianie testów jednostkowych oraz uruchamianie aplikacji.

## Struktura Projektu

- `app.py`: Główna aplikacja.
- `test_app.py`: Testy jednostkowe dla aplikacji.
- `requirements.txt`: Lista zależności.
- `Makefile`: Plik zawierający reguły do zarządzania projektem.
- `README.md`: Dokumentacja projektu.

## Wymagania

- Python 3.x
- `pip` - menedżer pakietów dla Pythona

## Instalacja

Aby zainstalować wymagane zależności, użyj poniższej komendy:

```bash
make install
